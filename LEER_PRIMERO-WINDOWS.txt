═══════════════════════════════════════════════════════════════════
INSTALACIÓN RÁPIDA
═══════════════════════════════════════════════════════════════════

1. Descomprime el archivo ZIP a una carpeta de tu elección

2. Entra a la carpeta GesMonth/

3. Haz doble clic en GesMonth.exe

¡Eso es todo! La aplicación iniciará automáticamente.

═══════════════════════════════════════════════════════════════════
REQUISITOS DEL SISTEMA
═══════════════════════════════════════════════════════════════════

- Windows 10 o superior (64-bit)
- 200 MB de espacio en disco
- 512 MB de RAM (mínimo)
- Resolución mínima: 1024x768

═══════════════════════════════════════════════════════════════════
PRIMEROS PASOS
═══════════════════════════════════════════════════════════════════

1. Al iniciar por primera vez, GesMonth creará automáticamente una
   base de datos SQLite en la misma carpeta del ejecutable.

2. El Dashboard mostrará estadísticas iniciales (todo en 0).

3. Ve a "Clientes" para agregar tus primeros clientes.

4. Luego ve a "Cuotas" para registrar pagos mensuales.

5. Usa "Reportes" para exportar información a Excel.

═══════════════════════════════════════════════════════════════════
SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════════════

PROBLEMA: Windows SmartScreen bloquea la aplicación
SOLUCIÓN: 
  • Haz clic en "Más información"
  • Luego en "Ejecutar de todas formas"
  • Esto es normal para aplicaciones no firmadas digitalmente

PROBLEMA: Antivirus marca el ejecutable como sospechoso
SOLUCIÓN: 
  • Es un falso positivo común con ejecutables PyInstaller
  • Agrega GesMonth.exe a la lista de excepciones de tu antivirus
  • El código fuente está disponible en GitHub para verificación

PROBLEMA: No se ven los estilos
SOLUCIÓN: 
  • Asegúrate de que la carpeta assets/ esté junto al ejecutable
  • No muevas GesMonth.exe fuera de su carpeta

PROBLEMA: Error al exportar a Excel
SOLUCIÓN: 
  • Cierra cualquier archivo Excel que tenga el mismo nombre
  • Verifica que tengas permisos de escritura en la carpeta

═══════════════════════════════════════════════════════════════════
CREAR ACCESO DIRECTO EN EL ESCRITORIO
═══════════════════════════════════════════════════════════════════

1. Haz clic derecho sobre GesMonth.exe
2. Selecciona "Crear acceso directo"
3. Arrastra el acceso directo a tu escritorio
4. (Opcional) Renómbralo a "GesMonth"

═══════════════════════════════════════════════════════════════════
CARACTERÍSTICAS PRINCIPALES
═══════════════════════════════════════════════════════════════════

- Dashboard con estadísticas en tiempo real
- Gestión completa de clientes (CRUD)
- Registro de pagos mensuales
- Control de cuotas por año/mes
- Reportes exportables a Excel
- Interfaz moderna con efectos de cristal (glassmorphism)
- Base de datos SQLite integrada
- Búsqueda y filtrado avanzado

═══════════════════════════════════════════════════════════════════
UBICACIÓN DE LA BASE DE DATOS
═══════════════════════════════════════════════════════════════════

La base de datos (gesmonth.db) se crea automáticamente en la misma
carpeta donde está GesMonth.exe.

Para hacer respaldo:
  • Copia el archivo gesmonth.db a otro lugar seguro

Para restaurar:
  • Reemplaza gesmonth.db con tu copia de respaldo

═══════════════════════════════════════════════════════════════════
DESINSTALACIÓN
═══════════════════════════════════════════════════════════════════

GesMonth no modifica el registro de Windows ni instala archivos
en otras ubicaciones. Para desinstalarlo:

1. Cierra la aplicación si está abierta
2. Elimina la carpeta completa GesMonth-v1.0.0-Windows/

Si quieres conservar tus datos:
  • Guarda el archivo gesmonth.db antes de eliminar la carpeta
