INSTALACIÓN RÁPIDA
═══════════════════════════════════════════════════════════════════

1. Descomprime el archivo ZIP a una carpeta de tu elección

2. Entra a la carpeta GesMonth/

3. Haz doble clic en GesMonth.exe

4. Si Windows SmartScreen bloquea la aplicación:
   • Haz clic en "Más información"
   • Luego en "Ejecutar de todas formas"
   • Esto es normal para aplicaciones no firmadas digitalmente
   • Todo el código fuente está disponible en GitHub para
     verificación y confianza en la seguridad de la aplicación:
     https://github.com/DIlanSG/GesMonth

¡Eso es todo! La aplicación iniciará automáticamente.

REQUISITOS DEL SISTEMA
═══════════════════════════════════════════════════════════════════

═══════════════════════════════════════════════════════════════════
PRIMEROS PASOS
═══════════════════════════════════════════════════════════════════

1. Al iniciar por primera vez, se mostrará la pantalla de login.
   Credenciales por defecto:
   Usuario: admin
   Contraseña: admin123
   (IMPORTANTE: Cambiar después del primer ingreso)

2. GesMonth creará automáticamente las bases de datos SQLite en la
   carpeta .data/ junto al ejecutable.

3. El Dashboard mostrará estadísticas en tiempo real.

4. Ve a "Clientes" para agregar tus primeros clientes.

5. Usa "Cuotas" para registrar pagos mensuales.

6. Consulta "Reportes" para ver estadísticas mensuales.

7. (Solo Superadmin) Ve a "Configuración" > "Historial de Cambios"
   para ver todos los logs del sistema.

═══════════════════════════════════════════════════════════════════
SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════════════

PROBLEMA: Antivirus marca el ejecutable como sospechoso
SOLUCIÓN: 
  • Es un falso positivo común con ejecutables PyInstaller
  • Agrega GesMonth.exe a la lista de excepciones de tu antivirus
  • El código fuente está disponible en GitHub para verificación

PROBLEMA: No se ven los estilos
SOLUCIÓN: 
  • Asegúrate de que la carpeta assets/ esté junto al ejecutable
  • No muevas GesMonth.exe fuera de su carpeta

PROBLEMA: Error al exportar a Excel
SOLUCIÓN: 
  • Cierra cualquier archivo Excel que tenga el mismo nombre
  • Verifica que tengas permisos de escritura en la carpeta

═══════════════════════════════════════════════════════════════════
UBICACIÓN DE LAS BASES DE DATOS
═══════════════════════════════════════════════════════════════════

Las bases de datos se crean automáticamente en la carpeta .data/:
  • gesmonth.db - Datos del negocio (clientes, pagos, cuotas)
  • users.db - Usuarios, sesiones y logs del sistema

Para hacer respaldo:
  • Copia toda la carpeta .data/ a otro lugar seguro
  • O usa la opción de respaldo desde Configuración > Mantenimiento
  • Para exportar logs: Configuración > Mantenimiento > Descargar Logs

Para restaurar:
  • Reemplaza gesmonth.db y/o users.db con tu copia de respaldo


Esta accion también se puede hacer con los usuarios, si quieres 
conservar los datos de usuarios guarda la carpeta .data completa

═══════════════════════════════════════════════════════════════════
DESINSTALACIÓN
═══════════════════════════════════════════════════════════════════

GesMonth no modifica el registro de Windows ni instala archivos
en otras ubicaciones. Para desinstalarlo:

1. Cierra la aplicación si está abierta
2. Elimina la carpeta completa GesMonth-v{{VERSION}}-Windows/

Si quieres conservar tus datos:
  • Guarda la carpeta .data antes de eliminar la carpeta
