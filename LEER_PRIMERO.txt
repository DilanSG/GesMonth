DESCRIPCIÓN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

GesMonth es un sistema profesional para la gestión de pagos 
y cuotas mensuales. Perfecto para pequeños negocios, academias,
gimnasios, o cualquier servicio con cobros recurrentes.


INICIO RÁPIDO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Doble clic en GesMonth.exe
2. La aplicación iniciará en pantalla completa
3. La base de datos se crea automáticamente
4. ¡Listo para usar!


 CARACTERÍSTICAS PRINCIPALES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

-Dashboard Inteligente
   • Resumen de métricas clave
   • Total de clientes y estado de pagos
   • Recaudación mensual y anual

-Gestión de Clientes
   • Alta, baja y modificación de clientes
   • Búsqueda rápida
   • Valor de cuota personalizado

-Control de Cuotas
   • Vista mensual organizada por años
   • Estados: Pagado, Impago, Con Deuda, Pendiente
   • Seguimiento de mora automático
   • Soporte para pagos parciales

-Registro de Pagos
   • Múltiples métodos de pago
   • Historial completo
   • Prevención de duplicados

-Reportes y Estadísticas
   • Filtros por mes y año
   • Estadísticas detalladas
   • Desglose por método de pago

-Configuración
   • Gestión de métodos de pago
   • Años de facturación
   • Respaldo de base de datos
   • Limpieza de datos


GUÍA DE USO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Navegación
──────────
Usa el menú lateral izquierdo para cambiar entre secciones:
• Dashboard: Vista general del sistema
• Clientes: Gestión de clientes
• Cuotas: Control mensual de pagos
• Reportes: Estadísticas y análisis
• Configuración: Ajustes del sistema

Agregar Cliente
───────────────
1. Ve a "Clientes"
2. Clic en "Agregar Cliente"
3. Llena el formulario
4. Guarda

Registrar Pago
──────────────
1. Ve a "Cuotas"
2. Busca el cliente
3. Clic en el mes correspondiente
4. Selecciona "Registrar Pago"
5. Ingresa el monto y método
6. Confirma

Marcar Impago
─────────────
1. Ve a "Cuotas"
2. Busca el cliente
3. Clic en el mes
4. Selecciona "Marcar como Impago"
5. Se iniciará el seguimiento de mora

Ver Estadísticas
────────────────
1. Ve a "Reportes"
2. Selecciona año y mes
3. Visualiza las estadísticas
4. Revisa el desglose por método de pago

Respaldar Datos
───────────────
1. Ve a "Configuración"
2. Sección "Mantenimiento"
3. Clic en "Respaldar Base de Datos"
4. Selecciona ubicación
5. Se creará un archivo .db de respaldo


IMPORTANTE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ NO mover o eliminar la carpeta "assets"
✓ Hacer respaldos periódicos de la base de datos
✓ La base de datos se guarda en: gesmonth.db
✓ Todos los datos son locales (no requiere internet)


SOLUCIÓN DE PROBLEMAS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Problema: La aplicación no inicia
└─ Solución: Verifica que la carpeta "assets" esté presente

Problema: No se ven los estilos
└─ Solución: Reinstala la aplicación completa

Problema: Pagos duplicados
└─ Solución: Ve a Configuración > Limpiar Pagos Duplicados

Problema: Error al guardar
└─ Solución: Verifica permisos de escritura en la carpeta


UBICACIÓN DE DATOS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Base de Datos: gesmonth.db (en la misma carpeta del ejecutable)
Respaldos: Ubicación que elijas al crear el respaldo
Reportes: Ubicación que elijas al exportar (futuro)
